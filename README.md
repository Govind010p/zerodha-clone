📈 Zerodha Clone – Full Stack Trading Platform
 
A full-stack stock trading web application inspired by Zerodha, built to simulate real-world trading functionalities including buying, selling, portfolio tracking, and real-time market updates.

🚀 Features
   🔐 User Authentication (JWT-based login/signup)
   📊 Real-time stock price updates using Socket.io
   💰 Buy & Sell stocks with dynamic pricing
   📁 Portfolio management (Holdings, Positions, Orders)
   ⭐ Watchlist to track favorite stocks
   📉 Live dashboard with market data
   📱 Responsive UI for desktop and mobile
